# 任务代号：全球运营商财报自动侦察与 BLM 分析 (V6.0 模板)

你好，Agent。这是一项复杂的自动化侦察任务。你需要读取指定的参数文件，然后调用浏览工具、文件下载与解析工具来执行分析。

## 1. 读取参数
首先，请读取用户指定的 **参数文件 (Parameters File)**。你需要从中提取以下关键信息：
- `[TIME_ANCHOR]`: 目标分析的统一日历时间段 (例如: "2023自然年")
- `[TARGET_COUNTRY]`: 目标锁定国家 (例如: "德国")
- `[TARGET_OP_NAME]` & `[TARGET_OP_URL]`: 主角运营商名称和财报页面 URL
- `[COMPETITOR_list]`: 竞争对手的名称和 URL 列表

## 2. 执行步骤流 (Agent Workflow)

### 阶段一：自主寻宝与数据采集
对参数文件中列出的每一个 URL 执行以下操作：
1.  **访问与扫描：** 使用浏览器打开 URL。
2.  **识别关键资产：** 在页面上寻找指向核心财报文件的链接。优先寻找：Annual Report (PDF), Full Year Results Presentation (PPT), Financial Data/KPIs (Excel/Spreadsheet)。
3.  **下载与解析：** 下载最有价值的文档并解析其内容。

### 阶段二：双重过滤清洗 (关键！)
对解析出的数据进行严格清洗：
1.  **时间对齐 (Time Alignment)：** 检查报告期是否与 `[TIME_ANCHOR]` (日历时间) 一致。如果是跨期财年，必须尽力折算为目标日历年的数据，或找到官方提供的日历年重述数据。如果无法精确对齐，必须在最终报告中标注口径差异。
2.  **地理位置摘取 (Geo Extraction)：** 对于跨国公司，仅摘取关于 `[TARGET_COUNTRY]` 的分部财务数据和管理层论述，丢弃其他国家数据。

### 阶段三：BLM 矩阵生成
利用清洗后的干净数据，生成一个 Markdown 格式的 BLM 对标矩阵报告。报告须包含：
- **战略意图横评：** 各家在目标市场的核心战略口号和投资重点。
- **硬核业绩对撞表：** 在统一日历时间内，各家在目标市场的营收、利润、移动用户ARPU、以及**产业数字化/云业务收入**的对比。

## 3. 输出要求
请将最终生成的 Markdown 报告保存在 `outputs/` 文件夹下，文件名为 `BLM_Report_[TARGET_COUNTRY]_[TIME_ANCHOR].md`。
