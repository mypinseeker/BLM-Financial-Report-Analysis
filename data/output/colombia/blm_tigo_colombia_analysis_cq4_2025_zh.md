# 执行摘要

## 一句话判断

**Tigo Colombia 在完成收购 Coltel（Movistar哥伦比亚）后，从"夹心层"运营商一跃成为频谱资源最丰富的运营商（265 MHz），拥有3-5年窗口期将频谱优势转化为市场份额和营收领先。**

## 3. 核心发现

### 看一：趋势 (PEST)
- 宏观环境总体有利：PEST天气"晴天"（11项因素中7项为机会）
- GDP增长1.8%，通胀5.5%（中等压力）
- CRC宣布2026年5G频谱拍卖（3.5 GHz + 26 GHz）
- 哥伦比亚数字化2030战略：宽带普及 + 5G路线图

### 看二：市场与客户 ($APPEALS)
- 季度电信市场总收入COP 9,340B
- Tigo在8项$APPEALS维度中领先3项，落后5项
- 识别出4个客户细分：大众预付费、后付费融合、大企业、中小企业

### 看三：竞争
- 竞争强度：**高**
- Claro以48.7%营收份额领先，但份额持续下降 (-1.1pp)
- WOM以35.3%的营收增速快速增长，但仍亏损
- **Coltel收购使市场从4家变为3家**——竞争格局根本性改变

### 看四：自身分析
- 健康状态：**稳定**
- EBITDA利润率35.1%，低于Claro (40.0%)但高于行业平均
- 固网宽带为最强增长引擎（CAGR +9.6%，加速增长）
- 光纤用户55万，同比增长29.5%

### SWOT综合
- 优势9 / 劣势5 / 机会6 / 威胁7
- 战略姿态：**防守型 (ST主导)**
- 优势大于劣势，但威胁多于机会

### 看五：机会 (SPAN矩阵)
- 30个战略机会：15个增长/投资、7个获取能力、4个收割、4个规避
- 50%的机会位于"增长/投资"象限——机会格局有利

## 3. 客户细分

### 3.1 大众预付费
- **规模**: 占移动用户~70%
- **未满足需求**: 低收入用户的可负担数据套餐；农村/半城区覆盖
- **痛点**: 流量快速耗尽；波哥大/麦德林网络拥堵
- **决策因素**: 每GB价格；社交媒体包；充值便利性

### 3.2 后付费融合用户
- **未满足需求**: 高质量融合套餐（移动+光纤+电视）；5G就绪套餐
- **痛点**: 大城市外光纤覆盖慢；定价结构复杂
- **决策因素**: 套餐价值；网速；终端优惠

### 3.3 大企业客户
- **规模**: ~1.6万客户（B2B前5%）
- **未满足需求**: SD-WAN和多云连接；农业/物流IoT方案
- **竞争空白**: Claro在大企业ICT占优；云和安全管理服务有机会
- **机会**: 管理连接+云服务助力数字化转型

### 3.4 中小企业客户
- **规模**: ~31.4万客户
- **未满足需求**: 简单一站式商用连接；可负担云方案
- **竞争空白**: 多数运营商缺乏简化的中小企业套餐
- **机会**: 简单一体化商业套餐+数字工具

## 频谱优势分析

| 维度 | Tigo | Claro | WOM | Tigo优势 |
| :--- | :--- | :--- | :--- | :--- |
| 总频谱 | 265 MHz | 200 MHz | 60 MHz | **+32.5%** vs Claro |

| 中频（1-3 GHz） | 215 MHz | 150 MHz | 50 MHz | **+43.3%** vs Claro |
| 频谱/用户 | 8.5 kHz/用户 | 6.2 kHz/用户 | 13.0 kHz/用户 | 频谱效率优势 |
| FDD频谱 | 205 MHz | 200 MHz | 60 MHz | 略优 |
| TDD频谱 | 60 MHz | 0 MHz | 0 MHz | **独占TDD** |

**关键洞察**：

1. **中频段压倒性优势**: Tigo在1-3 GHz中频段持有215 MHz，比Claro多43%。这是4G/5G容量的核心频段。
2. **低频段持平**: 700 MHz + 850 MHz合计50 MHz，与Claro持平，农村覆盖不吃亏。
3. **FDD/TDD结构差异**: Tigo是唯一拥有TDD频谱（2500 MHz B41, 60 MHz）的运营商，在高密度场景下拥有灵活的上下行配比优势。Claro的2500 MHz为B7 FDD，虽然带宽相同，但TDD在数据下载密集场景下更具效率。
4. **频谱效率**: Tigo合并后每用户频谱为8.5 kHz，优于Claro的6.2 kHz，意味着更好的用户体验空间。
5. **WOM困境**: 仅60 MHz全FDD频谱，且无2500 MHz高容量频段，长期网络能力受限。
6. **5G就绪**: 700 MHz的40 MHz持有量（双10 MHz FDD载波）为5G FDD低频广覆盖提供最佳基础，可同时部署DSS。
